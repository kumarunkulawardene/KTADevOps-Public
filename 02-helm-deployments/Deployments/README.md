# Scenario Deployments

This folder contains scenario-based Kubernetes manifests and Azure CLI snippets.

The scenarios are useful for understanding the raw Kubernetes objects behind the Helm charts or for testing a specific pattern.

## Scenario Types

- basic deployment
- Application Gateway Ingress Controller
- image pull configuration
- Azure AD / managed identity / Key Vault integration
- runtime configuration and database connection examples

Prefer the Helm charts for repeatable deployments, and use these files as reference or troubleshooting material.
